⚙ Configuração necessária

      
Configure as variáveis de ambiente no Vercel para conectar à API do SEI:

      
    
        
  1. Acesse Vercel Dashboard → seu projeto → Settings → Environment Variables
    2. 
        
  2. Adicione: SEI_BASE_URL, SEI_USUARIO, SEI_SENHA, SEI_ORGAO
    3. 
        
  3. A URL base está no QR Code da barra lateral do SEI (menu esquerdo, rodapé)
    4. 
        
  4. Faça Redeploy após salvar as variáveis
    5. 
      

      
Veja .env.example no repositório para referência.
